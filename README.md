# Library Management System in C++

## Project Description

A console-based Library Management System developed using C++ with file handling.

## Features

* Add Book
* Display Books
* Search by Title
* Search by Author
* Issue Book
* Return Book

## Technologies Used

* C++
* Object-Oriented Programming
* File Handling (`fstream`)

## Files Used

* `books.txt`
* `temp.txt`
